# Sustainability Reports — NEXA

Relatório de Sustentabilidade de **Nexa SGOIC, S.A.**, sociedade gestora de organismos de investimento coletivo regulada pela CMVM (Portugal).

## Edição Web 2025

`index.html` — long-scroll editorial bilingüe (PT / EN), single-file (HTML + CSS + JS inline), sin framework.

Tokens en `nx-tokens.css`, decisiones en `BRAND_GUIDELINES.md`, design system en `design-system.html`.

### Probar localmente

```
python -m http.server 8765
# abrir http://localhost:8765/
```

### Estructura

```
sustainability-reports-NEXA/
├── index.html                  # Edição web (long-scroll editorial)
├── nx-tokens.css               # Tokens de diseño (single source of truth)
├── design-system.html          # Specimen visual del sistema
├── BRAND_GUIDELINES.md         # Decisiones numeradas R1–R25
├── brand/                      # Kit oficial (logos + Manual de Normas)
├── assets/brand/               # Logos en variantes adicionales
├── data/                       # 16 JSONs estruturados del relatório
├── docs/
│   ├── DESIGN_BRIEF.md         # Brief original a Claude Design
│   └── HANDOFF_TO_CLAUDE_CODE.md
├── pdfs/RS_NEXA_2025.pdf       # PDF source (Adobe InDesign export)
└── uploads/                    # Fotos, fuentes, referencias
```

## Relatório 2025

- **Período**: 1/01/2025 – 31/12/2025 (comparativo con 2024)
- **Ativos sob gestão**: 198 M€ — 18 ativos imobiliários (incluyendo landbank)
- **Cidades**: 8 — Portugal
- **Colaboradores**: 10
- **Fundos**: Icon SIC, Fuse Valley FII, Castro Capital SIC
- **Primeira certificação LEED Gold®**: edifício La Movida
